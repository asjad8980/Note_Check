A clean, lightweight note-taking application built for Android. This app focuses on speed, modern design, and user efficiency.

## ✨ Features
* **Dynamic Theming:** Automatically adapts its colors to match your phone's wallpaper (Material You).
* **Smart Search:** Instantly find any note by typing in the search bar.
* **Multi-Select Mode:** Long-press to select multiple notes for batch deletion or marking as "Done."
* **Auto-Titling:** Automatically sets the note title based on your first line of text.
* **Empty State:** Friendly UI guidance when you have no notes yet.
